Guia de Creacion de Proyectos para Angular:
Antes de todo "ng" es el diminutivo de angular y el comando que usaremos a traves de la consola de comandos

Creacion de Proyecto | ng new "proyecto" |
La Creacion del Proyecto incluye Git

Abre el Proyecto en el navegador , actualizandolo cada vez que se cambian los archivos | ng serve |

Tambien sirve Abrir el Proyecto | npm start |

Guia QuickStart :
git clone https://github.com/angular/quickstart.git quickstart
cd quickstart
npm install
npm start

Para borrar los ficheros no necesarios usar estos comandos
for /f %i in (non-essential-files.txt) do del %i /F /S /Q
rd .git /s /q
rd e2e /s /q
